# morethan Brand Kit v1.1

This directory contains the approved exported brand kit for the current mark.

## Usage Rules

- Use the icon mark for small-size surfaces such as app icons, favicons, and compact UI placements.
- Use the horizontal or stacked lockups for brand-first contexts.
- Treat the guidelines PDF and design-token JSON as the reference inputs for downstream usage; do not duplicate asset inventories in this README.
