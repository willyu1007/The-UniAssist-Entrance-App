# 05 Pitfalls (do not repeat)

This file exists to prevent repeating mistakes within this task.

## Do-not-repeat summary (keep current)
- 不要把高风险动作偷渡到 `publish`。
- 不要让 runtime/connector 成为 secret 或 scope 的主权持有者。
- 不要只靠 UI 权限而没有正式 governance request。
- 不要再造一套和 `ApprovalRequest / ApprovalDecision` 并行的治理审批账本。
- 不要一开始把 policy 拆成过多专属对象。
