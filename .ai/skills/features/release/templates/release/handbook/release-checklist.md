# Release Checklist

## Pre-Release

- [ ] All tests passing
- [ ] Documentation updated
- [ ] release/CHANGELOG.md updated
- [ ] Version bumped in package files
- [ ] Dependencies updated
- [ ] Security audit passed

## Release

- [ ] Tag created
- [ ] Release notes written
- [ ] Artifacts built
- [ ] Published to registry (npm, pypi, etc.)
- [ ] Docker images pushed

## Post-Release

- [ ] Announce release (blog, social, etc.)
- [ ] Update project board
- [ ] Close milestone
- [ ] Notify stakeholders
