# Example: Triage quick map

- `401`:
  - auth token/cookie not present or not recognized
  - token expired or invalid signature
  - middleware not executed (order issue)

- `403`:
  - permission rule denies
  - wrong identity/tenant context
  - mismatch between UI expectations and backend rule

- `404`:
  - wrong prefix or path parameter shape
  - method mismatch
  - route not registered or shadowed by another route
