# Handoff checklist

- [ ] Current status and next steps are clear.
- [ ] Key decisions and tradeoffs are recorded.
- [ ] Important pitfalls and do-not-repeat notes are recorded in `05-pitfalls.md`.
- [ ] Known issues and risks are documented.
- [ ] Verification steps are listed and current status is recorded.
- [ ] Any required follow-up work is assigned or clearly described.
